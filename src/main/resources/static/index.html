Bienvenido a la API de Gestión de Tareas

Esta API es el núcleo de un proyecto para una aplicación que gestiona una lista de tareas. El sistema permite a los usuarios crear, leer, actualizar y eliminar tareas, brindando una solución completa para la administración de actividades.

Tecnologías Utilizadas

  • MongoDB: Base de datos NoSQL para el almacenamiento de información de forma escalable y flexible.
  • JWT (JSON Web Tokens): Mecanismo de autorización y autenticación para asegurar que solo los usuarios autorizados puedan acceder a las funcionalidades protegidas de la API.
  • Spring Boot con Kotlin: Framework para el desarrollo rápido y eficiente de aplicaciones basadas en microservicios.

Características Principales

  • Gestión completa de tareas (crear, leer, actualizar, eliminar).
  • Seguridad y autorización mediante JWT.
  • Interacción con una base de datos MongoDB para el almacenamiento persistente.

Este proyecto sirve como una base sólida para aplicaciones de gestión de tareas, permitiendo la integración con diversos frontends y ampliaciones futuras.
